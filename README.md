# VLESS VPN для apelsin.space

### Настройка удаленного сервера
Для настройки VLESS VPN server на вашей удаленной машине выполните скрипт `server-setup.sh` при помощи команды (на MacOS, если у вас оболочка zsh):
```
./server-setup.sh
```

# План разработки проекта
1. Написать скрипт для запуска с локального компьютера и настройки удаленного сервера на работу с VPN + шаблон файла с переменными окружения.
2. Написать Backend для добавления новых пользователей к VPN (Kotlin Backend, DB).
3. Написать Frontend (Telegram Mini-app / React App на самом сервере).
4. Добавить возможность оплаты VPN (для этого надо хранить даты последней оплаты, регистрации и т. д.).
